# 🦾 같이할래? (제목 미정)

## 👨‍💻 프로젝트 소개
 - 같이할래 프로젝트는 위치기반 또는 사용자가 원하는 지역에서 자신의 취미에 맞는 동호회를 찾거나, 자신이 동호회 리더가 되어 함께 취미를 공유하고 새로운 만남을 가질 수 있는 커뮤니티를 제공합니다.
